Milestone: Initial Setup
Epic: Resource Center
Feature: FAQs Section
User Story: As a user, I want to read FAQs so that I can find quick answers to common questions.
Acceptance Criteria:
1. Users can access the FAQs section from the main menu.
2. Users can search for specific questions within the FAQs.
3. Users can submit a question if it is not answered in the FAQs.

Issues to Address:
- Ensure that the FAQs section is easily accessible from the main menu.
- Verify that the search functionality works correctly, including handling of edge cases such as no results found.
- Implement a submission form for unanswered questions, ensuring validation for empty submissions.
- Consider user experience for mobile devices when accessing the FAQs section.
